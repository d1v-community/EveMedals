# EVE Medals 1 分钟极简答辩版

更新时间：2026-03-31

## 版本一：最稳主版

大家好，我们做的是 `EVE Medals`。

一句话讲：

> `EVE Medals` 把 `EVE Frontier` 玩家真实发生过的行为，变成可验证的勋章、Warrior 档案和可分享卡片。

它想解决的问题很直接：`EVE Frontier` 玩家明明投入了大量探索、战斗、建设和协作，但这些荣誉缺少一种可验证、可展示、可传播的表达方式。

我们现在已经把一条主链路跑通了：

`行为索引 -> 成就判断 -> 勋章领取 -> Warrior 展示 -> 分享卡片`

也就是说，我们会先把玩家的 Frontier 行为整理成成就状态，再把它变成可领取的勋章、可展示的 Warrior 档案，以及可分享的社交卡片。

它不是单纯发 NFT 图片，而是在把玩家真实发生过的行为，转成可验证、可展示、可传播的荣誉资产。

当前我们已经完成了成就验证、勋章展示和分享能力。现在展示的是一个 `working product loop`，下一步会继续补齐更可信的环境对接、验证闭环和增长归因。

## 版本二：更偏评委口径

`EVE Medals` 是一个面向 `EVE Frontier` 的成就验证与传播产品。

它的核心不是“发一个 NFT”，而是把玩家真实发生过的行为，通过索引、规则和链上持有证明，变成勋章、Warrior 页面和可分享卡片。

所以我们做的其实是一层新的荣誉表达方式：

- 有行为依据
- 有状态判断
- 有链上承载
- 有分享入口

当前主链路已经打通，下一阶段重点是把它继续补成更可信、也更可量化的身份与传播基础设施。
